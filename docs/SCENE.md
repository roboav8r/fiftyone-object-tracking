# Scene

The **Scene** tab of the `ObjectTracking` panel is a per-scene
bird's-eye-view of object trajectories, rendered live from the source
tracking dataset with a timeline scrubber, a base ↔ world frame toggle,
per-instance presence rows, and pan / zoom.

> Source dataset schema: see [DATASET_SCHEMA.md](DATASET_SCHEMA.md).

## Browse a single scene

On the source tracking dataset (not the trajectories one), open the
`ObjectTracking` panel from the panel `+` menu and select the **Scene**
tab. Pick a scene, scrub the timeline, and toggle between base- and
world-frame views.
